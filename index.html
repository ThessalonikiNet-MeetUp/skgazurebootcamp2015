Azure Bootcamp

Global Azure Bootcamp is an event that is organized the last years by local communities over the globe. It is an one day of deep dive class to help people get up to speed on developing Cloud Computing Applications for Azure. It is an excellent opportunity to get to know Microsoft's Cloud platform even if you have no experience with it.

Subjects

Introduction to Microsoft Azure

During the event an extensive introduction about Microsoft Azure will take place. We will try to cover most of the platform's popular features.

Going beyond the basics

After the introduction we will present some more advanced characteristics. On top of this we are going to demonstrate how a real application on Microsoft Azure looks like.

Answer to your questions

We can't guarantee that we will be able to answer all of your questions but we will try at least. During the event you will be able to ask all different kinds of questions regarding Microsoft Azure.

Sessions

  • 9:30 - 10:00

    Introduction to Microsoft Azure

    During this short session we will go through a short introduction about Microsoft's cloud platform, Azure.

  • 10:15 - 11:15

    Azure IaaS

    You can use Azure for a new web app, or to design a new database, but what happen if you have already a legacy system that must communicate with your new web app, or you need to extend a "not Azure ready system"? In this session we will create a virtual machine running Linux os and create a mysql instance. After that we will create the virtual network to wire our web app with this legacy system.

  • 11:30 - 12:45

    Azure App Service API apps

    The API Apps support within Azure App Service provides additional support that enables you to easily create, consume and call APIs - both APIs you create (using a framework like ASP.NET Web API or the equivalent in other languages) as well as APIs from other SaaS and cloud providers. During the presentation we will explore the fundamentals of creating, calling & consuming REST APIs using ASP.NET Web API on Azure. We will also examine a test case of using an API from a third-party cloud provider within Azure.

  • 13:30 - 14:30

    Azure Mobile Services

    In this session we will explore how you can use Azure Mobile Services with your mobile application to add an authentication system using some of the most famous provides like Facebook / Google / Microsoft, relieving you from the pain of setting up your own authentication system

  • 14:45 - 15:45

    Leveraging Azure Mobile Services for Better Mobile Apps

    Mobile development has many challenges: Many platforms, high availability, security, large amount of data, short development cycles, app monitoring and more. We are going to show how Azure Mobile Services can help to answer these challenges. We are going to cover, from setting the azure environments and how we can access that infrastructure from various mobile clients. We are going to achieve live communication with push notifications. We are also going to show how the very new azure app services bundle with mobile development.

  • 16:00 - 17:00

    Polyglot persistence in Microsoft Azure

    Traditional relational databases have been conquer the world for over thirty years. Even though they are still the dominant solution when it comes to store data there are other more suitable options for certain scenarios. The term noSQL databases is well known but misused a lot. During the session we will try to present the various options the Microsoft Azure provides for polyglot persistence such as SQL Azure, Table and Blob storage, Redis Cache and DocumentDB. We are going to describe which choice is better for various scenarios and give examples based on a real application.
