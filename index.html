Azure Bootcamp

Global Azure Bootcamp is an event that is organized the last years by local communities over the globe. It is an one day of deep dive class to help people get up to speed on developing Cloud Computing Applications for Azure. In addition to this great learning opportunity we will have another set of hands on labs.

Subjects

Introduction to Microsoft Azure

During the event an extensive introduction about Microsoft Azure will take place. We will try to cover most of the platform's popular features.

Going beyond the basics

After the introduction we will present some more advanced characteristics. On top of this we are going to explore how a real application on Microsoft Azure looks like.

Building a real application

The ultimate goad of the event is to delve into writing code for the Azure. So half of the time of the event will be a workshop where we will build an application that uses the platform.
